1. go to https://canvas.unab.cl/courses/[course-id]/external_tools/187
2. this will dispatch a xhr request to https://applications.zoom.us/api/v1/lti/rich/meeting/upComing/COURSE/all?page=1&total=0&storage_timezone=America/Santiago&client_timezone=America/Santiago&storage_language=en-US&client_language=en-US&lti_scid=a943eade38ad3ec274df131111492b54a2b5d148ea1d6b1173746e724f760a57 scid is the course id and can be used later to get the list of recordings
3. the request with scid and cookies to `https://applications.zoom.us/api/v1/lti/rich/recording/COURSE?startTime=&endTime=2025-09-16&keyWord=&searchType=1&status=&page=1&total=0&lti_scid=a54cecfe2cba47c7701529c68fe2e09a2fc9737ec6b7e905de6c72b6b3bfea6a` or `https://applications.zoom.us/api/v1/lti/rich/recording/COURSE?startTime=&endTime=2025-09-16&keyWord=&searchType=1&status=&page=1&total=0&lti_scid=a943eade38ad3ec274df131111492b54a2b5d148`, will return a list of the recordings. this will be used to iterate and download the recordings.
4. using the same cookies make a request to https://applications.zoom.us/api/v1/lti/rich/recording/file?meetingId=[meeting-id]&lti_scid=[scid], iterating over the previously saved file with the recordings. save all the responses.
5. iterate over the previous step responses and look for result.recordingFiles.playUrl object, this link is the recording file of the meeting, save it into a list.
6. iterate the files and use ffmpeg to rip it to local.
